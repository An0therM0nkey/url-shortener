  • POST links_pairs/

    Description: creates a links_pair for "original_url"

    Request body parameters:
    Name Required Default Type Description
    original_url True string url you want to shorten
    days_limit False 90 int period, in which short url will be available

    Response: single links pair object

  • GET links_pairs/

    Description: returns all links_pairs

    Response: list of links pair objects

  • GET links_pairs/<pair_id>

    Description: returns a links_pair with id="pair_id"

    Get parameters:
    Name Type Description
    pair_id int links pair id

    Response: single links pair object

  • DELETE links_pairs/<pair_id>

    Description: deletes a links_pair with id="pair_id"

    Get parameters:
    Name Type Description
    pair_id int links pair id

    Response: single links pair object

  • Links pair object:
    Name Type
    date_created string
    original_url string
    id int
    days_limit int
    short_url string
